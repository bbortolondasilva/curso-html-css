Trabalhando com Listas:

Listas Ordenadas:

Com a tag <ol> podemos criar uma lista ordenada, podendo mudar o tipo do índice (1, 2, 3, I, II, III, A, B, C, etc.), e adicionar itens à esta lista com a tag <li>.

        
Exemplo:
    <ol>
        <li>Acordar;</li>
        <li>Tomar café;</li>
        <li>Tomar banho;</li>
        <li>Escovar os dentes;</li>
        <li>Ir para o trabalho.</li>
    <ol>

Ex. Lista:

  1. Acordar;
  2. Tomar café;
  3. Tomar banho;
  4. Escovar os dentes;
  5. Ir para o trabalho.

Listas não Ordenadas:

Com a tag <ul> podemos criar uma lista ordenada, podendo mudar o tipo do índice (circle, disk e square), e adicionar itens à esta lista com a tag <li>.

  • Acordar;
  • Tomar café;
  • Tomar banho;
  • Escovar os dentes;
  • Ir para o trabalho.
